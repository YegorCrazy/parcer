Это программа для парсинга сайта с объявлениями об аренде койкоместа с последующим сохранением данных в базу данных SQL.

Необходимые библиотеки: 
- lxml;
- bs4;
- PyMySQL;
- cryptography.

Каждую из них можно установить командой "pip3 install (имя_библиотеки)" в командной строке.

config.json - файл настроек. В нем:
usr - имя пользователя, от которого выполняется вход на сервер
pwd - пароль для учетной записи этого пользователя
database - название базы данных, к которой добавляются таблицы с данными. Ее нужно предварительно создать.
